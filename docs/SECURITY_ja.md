# セキュリティ

## 概要

fileview (fv) はユーザーの権限で動作するファイルブラウザです。

## セキュリティモデル

- 現在のユーザーのファイル権限で動作
- 権限昇格やネットワーク操作なし
- ファイル操作はユーザー自身のファイルシステムに影響

## --on-select コールバック

`--on-select` オプションはシェルコマンドを実行します。セキュリティ上の考慮事項:

- コマンドはあなたのシェルと権限で実行される
- パスはシングルクォートでエスケープされる
- 信頼できないコマンド文字列では使用しないこと
- ターミナルで手動でコマンドを実行するのと同等

### 安全な使用法

```bash
fv --pick --on-select "code {}"      # エディタで開く
fv --pick --on-select "cat {}"       # ファイルを表示
```

### 安全でない使用法 (避けるべき)

```bash
fv --on-select "$UNTRUSTED_VAR {}"   # 信頼できない入力を使用しない
```

## 脆弱性の報告

セキュリティの問題は GitHub Security Advisories またはメールで報告してください。
